# Inventory Management System

A full stack Inventory Management System built using Django, MySQL, HTML, CSS and JavaScript.

## Features
- User Authentication
- Product & Category Management
- Purchase Management
- Sales & Billing
- Payment Integration
- Reports & Analytics
- Near Expiry Offers

## Tech Stack
- Python
- Django
- MySQL
- HTML/CSS/JS

## How to Run
1. Install requirements
2. Configure database
3. Run migrations
4. Start server
